# alibekov.pro — лендинг тренинга «Хватит ли вам денег на всю оставшуюся жизнь?»

Статичный одностраничник под семинар-тренинг Тимура Алибекова.
Один файл — `index.html`, без сборки и зависимостей (шрифты Google Fonts по CDN).

**Стиль:** «Деловая газета» (editorial) — кремовая бумага, засечки, тонкие линии, акцент salmon/охра.
**Доступность:** свёрстано под скринридер (Тимур незрячий) — семантика, skip-link, aria-labels,
декор скрыт от озвучки, контраст AA, таймер скрыт от озвучки (вместо него текстовая дата),
уважается `prefers-reduced-motion`.

## ⚠️ Перед публикацией — вставить номер WhatsApp
В `index.html`, внизу в `<script>` (блок «НАСТРОЙКИ»):

```js
var WHATSAPP_NUMBER = "7XXXXXXXXXX"; // межд. формат, без + и пробелов, напр. 77011234567
var WHATSAPP_TEXT   = "...";          // автотекст заявки (можно оставить)
var EVENT_ISO       = "2026-06-22T21:00:00+05:00"; // дата/время старта (Алматы)
```
Пока номер не вставлен — кнопки просто скроллят к секции записи (сайт не «ломается»).

## Опционально
- **Фото Тимура** — сейчас плейсхолдер «ТА». Заменить блок `.photo` на `<img src="timur.jpg" alt="Тимур Алибеков">`.
- **og.png** — картинка превью для репостов в мессенджерах (1200×630), положить рядом.

## Деплой на alibekov.pro

### Вариант A — GitHub Pages БЕЗ Cloudflare (проще, HTTPS бесплатно от GitHub)
1. Репозиторий → загрузить `index.html` (+ `CNAME` со строкой `alibekov.pro`).
2. Settings → Pages → Source: `main` / root.
3. У регистратора домена в DNS:
   - `A` для apex → `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   - `CNAME` для `www` → `<username>.github.io`
4. Settings → Pages → включить **Enforce HTTPS** (после выдачи сертификата, ~15–30 мин).

### Вариант B — через Cloudflare (если домен уже там / нужен CDN)
- В Cloudflare DNS те же `A`-записи на IP GitHub Pages (можно «DNS only», серое облако — надёжнее для Pages).
- SSL/TLS mode: **Full**.
- Cloudflare НЕ обязателен — нужен только если хочешь их прокси/кэш/аналитику.

### Вариант C — обычный хостинг с FTP (если домен на ps.kz/Plesk)
- Просто залить `index.html` в корень сайта. Cloudflare не нужен.
